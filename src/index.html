Información

¿Qué es el nivel de cifrado u offset?

El cifrado césar es una de las técnicas más simples para cifrar un mensaje.
Es un tipo de cifrado por sustitución, es decir que cada letra del texto original
es reemplazada por otra que se encuentra un número fijo de posiciones (desplazamiento)
más adelante en el mismo alfabeto. A este desplazamiento se le llama offset.

¿Qué necesito para descifrar un mensaje?

Al elegir un offset o nivel de cifrado, este indica cuántas posiciones queremos
que el cifrado desplace cada letra del mensaje. Entonces, si envías tu mensaje secreto
pero no le dices a tu cómplice el número de offset, no podrá decifrarlo.
